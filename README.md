# CVSCode

![](images/banner.png)

CVSCode is a simple scripting language that is built with TypeScript, made by CVS0.

# Features:
* Let / Const variables
* Frequently updated
* Fast language
* Detailed error logs
* Similar syntax to JavaScript / TypeScript
